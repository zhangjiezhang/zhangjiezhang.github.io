Liunx下的操作

linux下安装sublime 3

  • wget https://download.sublimetext.com/sublime_text_3_build_3126_x64.tar.bz2：获取压缩文件（64位）.
  • tar jxvf sublime_text_3_build_3126_x64.tar.bz2：解压.
  • mv sublime_text_3 /usr/local/sublime：移动到指定位置.
  • ln -s /usr/local/sublime/sublime_text /usr/local/bin/sublime_text：添加软链接.
  • sublime_text：运行sublime 3.
  • cp /usr/local/sublime/sublime_text.desktop /usr/share/applications：创建桌面快捷方式.
  • vim /usr/share/applications/sublime_text.desktop：修改桌面配置文件.
  • Exec=/usr/local/sublime/sublime_text %F：//sublime_text路径地址
  • Icon=/usr/local/sublime/Icon/48x48/sublime-text.png ：//sublime图标icon路径地址
  • Exec=/usr/local/sublime/sublime_text -n ：//打开新窗口路径
  • Exec=/root/project --command new_file ：//创建新文件的命令路劲
  • 应用程序 >编程 > Sublime Text”右键”将此启动器添加到桌面”来启动sublime text 3

linux下安装WebStorm

  • wget https://download.jetbrains.8686c.com/webstorm/WebStorm-2018.2.3.tar.gz：获取源码.
  • tar zxvf WebStorm-2018.2.3.tar.gz：解压.
  • mv WebStorm-182.4323.44 /usr/local/webstorm：移动到指定位置.
  • ln -s /usr/local/webstorm/bin/webstorm.sh /usr/local/bin/webstorm：添加启动软连接.
  • webstorm：运行webstorm

linux下安装vs code

  • sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc：导入下载源.
  • sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl= https://packages.microsoft.com/yumrepos/vscode\nenabled= 1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'：执行脚本.
  • yum check-update：检查更新.
  • sudo yum install code：安装.
